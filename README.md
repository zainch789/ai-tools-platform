
# AI Tools Platform

A modern, fast, and SEO-optimized AI tools platform built with Next.js, TypeScript, Tailwind CSS, and Supabase.

## 🚀 Project Vision

Build a production-ready AI platform offering 100+ free AI-powered tools for:

- Writing
- Marketing
- Business
- Education
- Coding
- Productivity
- YouTube
- Social Media
- Legal
- Creative Work

The platform will focus on speed, user experience, SEO, and scalability.

---

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- Vercel
